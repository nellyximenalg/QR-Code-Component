# 🧩 Proyecto: Componente QR Code

Este proyecto consiste en el desarrollo de un **componente de Código QR** utilizando **Astro** y **Tailwind CSS**.  
El objetivo es aplicar los conocimientos sobre **componentes**, **maquetación**, **estilos responsivos** y **utilidades CSS** para construir un diseño limpio, moderno y adaptable a diferentes dispositivos.

---

## 📖 Descripción general
Este proyecto consiste en el desarrollo de un componente QR Code que muestra una tarjeta con un código QR, un título y una pequeña descripción.

El sitio fue desarrollado utilizando Astro para organizar el proyecto en componentes y Tailwind CSS para los estilos. También se aplicó diseño responsivo, para que el componente se vea correctamente en diferentes tamaños de pantalla.
